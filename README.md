##Project Description

This project is a dynamic Cart Items Web App built using JavaScript. It allows users to add items to a list by typing into an input field and clicking a button.

The application demonstrates how to capture user input, create elements dynamically, and update the webpage in real-time using DOM manipulation. 

##Features

Add items dynamically to the cart/list
⌨️ Takes real-time user input
🔄 Clears input field after adding item
📋 Displays items instantly without page reload
🧩 Simple and reusable logic
⚡ Fast and interactive UI 

##Technologies Used

HTML5 – Structure (input field & container)
CSS3 – Basic styling (optional)
JavaScript (ES6) – DOM manipulation & event handling 

##Installation Steps

1. Clone the repository git clone https://github.com/PANJAMYNASAIRAM/Add-to-cart.git
2. Go to project folder "cd Add-to-cart"
3. Install dependencies "npm install"
4. Start the project "npm start"

##How It Works
User enters text in the input field
On button click, the sai() function is triggered
A new <li> element is created
Input value is added as text
Item is appended to the list
Input field is cleared for next entry
